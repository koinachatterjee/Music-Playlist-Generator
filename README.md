# 🎵 Music Playlist Generator

Create personalized music playlists based on user preferences and listening patterns.

## 📌 Project Info
- **Batch:** June 2026 — AI InternsElite
- **Type:** Minor Project
- **Tech Stack:** Python, Pandas, Scikit-learn, Spotify API

## 📊 Features
- Create personalized playlists
- Song recommendation based on listening history
- Collaborative filtering for similar user preferences
- Spotify API integration for music metadata
- Dynamic playlist generation and updates

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Install the required libraries
3. Add your Spotify API credentials
4. Run all cells from top to bottom
5. Generate personalized playlists
